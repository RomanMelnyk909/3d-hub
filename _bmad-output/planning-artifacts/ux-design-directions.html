Evaluation Guide — What to look for

Layout & Hierarchy

Which card size and grid density feels right? D1 & D4 maximize browsability. D3 puts more info on each card. D5 suits power users with filters.

Nav & Green Usage

D1/D2/D4/D6: white nav, green as accent only. D3: full green header — stronger brand presence. D6: green banner draws attention to creator CTA. Which feels right?

Metadata Visibility

D3 shows print metadata (layer height, infill, supports, filament) directly on cards. Others show it on the model page only. Does D3's approach help consumers evaluate faster?

Featured / Editorial

D2 has a prominent featured hero section. Others go straight to grid. Does editorial curation add value for cold-start (50 models), or does a clean grid work better?

Creator Emphasis

D6 puts the Upload CTA front-and-center with a banner. Others keep it in the nav. How prominent should creator onboarding be for V1?

Search Placement

D4 makes search the homepage hero. Others put it in the nav. Given most users arrive via SEO on model pages, how important is homepage search prominence?
